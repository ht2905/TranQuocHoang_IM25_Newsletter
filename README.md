### Environment Setup

The project environment is managed using **`renv`**, ensuring that the exact R package versions used to generate the final output are preserved.

1.  **Restore the Environment:** To install the exact package dependencies, open the project in R and run:
    ```r
    renv::restore()
    ```
    This will use the **`renv.lock`** file to set up the necessary environment, including the **`apaquarto`** package.

2.  **Render the Document:** Once the environment is restored, the final Word document (`.docx` file) can be regenerated by running the following Quarto command (Befehl):
    ```bash
    quarto render TranQuocHoang_IM25_Newsletter.qmd --to apaquarto-docx
    ```
    The final submission document is generated in **APA 7th Edition format** for Microsoft Word.
